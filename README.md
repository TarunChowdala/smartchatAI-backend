# 🧠 SmartChat AI – Backend

The backend service for **SmartChat AI**, an intelligent chat platform that integrates **Generative AI** and **RAG (Retrieval-Augmented Generation)** to enable:
- Real-time conversations
- Document-based Q&A
- Resume analysis

Built with **FastAPI** and **Python**, this backend is designed for speed, scalability, and seamless AI integration using **LangChain** and **Gemini API**.  

---

## 🚀 Features
- **AI-Powered Responses** – Generate natural, human-like replies with Gemini API.
- **RAG-Based Document Search** – Retrieve context from uploaded files for better answers.
- **Resume Analysis** – Extract insights and evaluate resumes.
- **RESTful API** – Clean, structured API endpoints for the frontend.
- **Scalable Architecture** – Designed to handle real-time interactions efficiently.

---

## 🛠 Tech Stack
- **Framework:** FastAPI (Python)
- **AI Integration:** Gemini API, LangChain
- **Document Processing:** RAG (Retrieval-Augmented Generation)
- **Authentication:** Firebase (via API calls)
- **Other Libraries:** Uvicorn, Pydantic, etc.
